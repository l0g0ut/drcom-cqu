# drcom-cqu
Drcom for CQU

    填上以下信息：
    username=''                     # student id
    password=''                     # password
    host_ip = ''                    # host ip in local area network
    host_name = ''                  # your host name
    mac = 0x000000000000            # mac address
    host_os = ''                    # host operating system
